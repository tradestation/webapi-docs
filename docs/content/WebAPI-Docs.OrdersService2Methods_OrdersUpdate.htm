TradeStation WebAPI

Orders Service : Update Order

Summary:¶

Resource for updating an open order. You cannot update an order that has been filled.

Cancel Replace Rules
Original order type Fields to updateCan only change order type to
Limit OrdersQuantity, Limit PriceMarket
Stop OrdersQuantity, Stop PriceMarket
Stop Limit OrdersQuantity, Stop Price, Stop LimitMarket
Stop Market Trailing Stop OrdersQuantity, Trail Amount

Details:¶

  • Method: PUT
  • Path: /orders/{orderid}
  • URI Parameters:
    • {orderid}
  • Request Body
    • Subset of Order object serialized in JSON or XML. Required fields:
      • AccountKey
      • OrderID
      • OrderType
      • Quantity
      • Price field(s) used in original order
      • Trailing Stop parameters to change

JSONP:¶

  • URI Parameters
    • {orderid}
  • Query String Parameters
    • data = URL Encoded JSON serialized Order object
    • callback = jQuery method to callback
    • oauth_token = AccessToken
  • URL: /jsonp/orders/{orderid}/cancelreplace?data=EncodedSerializedOrderObjectGoesHere&callback=jQuery17105945590266492218_1328295375410&_=1328295376318&oauth_token=AccessTokenGoesHere

  • Authentication: Requires a valid access token.

Returns:¶

Update Order result object

Errors:¶

  • 400 | {"Message":"Not an open order.","StatusCode":400}
  • 400 | {"Message":"Invalid account.","StatusCode":400}
  • 400 | {"Message":"Missing order type.","StatusCode":400}
  • 400 | {"Message":"Missing order quantity.","StatusCode":400}
  • 400 | {"Message":"Missing order limit price.","StatusCode":400}
  • 401 | Unauthorized
  • 5xx | Unknown internal service error. Contact TradeStation.
