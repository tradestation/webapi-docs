TradeStation WebAPI

Data Service : HotLists

Summary:¶

Snapshot hotlist response.

Details:¶

  • Method: GET
  • Path: /data/hotlist/{hotlist id}
  • URI Parameters:
    • {hotlist id}=id that maps to a given permutation of {Category, Exchange, Activity}

  • Authentication: Requires a valid access token.
  • Authorization: Requires the MarketData Scope

  • Returns: A collection of Quote objects

Errors:¶

  • 401 | Unauthorized
  • 404 | Not Found (Hotlist id's are case sensitive)
  • 5xx | Unknown internal service error. Contact TradeStation.
