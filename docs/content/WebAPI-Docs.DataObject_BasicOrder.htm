TradeStation WebAPI

DataObject : BasicOrder Request
Requires order data formatted as json or xml.

Example JSON ¶

{
"AccountKey":"123456",
"AssetType":"EQ",
"Duration":"DAY",
"GTDDate":"",
"LimitPrice":"10.00",
"OrderID":"",
"OrderType":"Limit",
"Quantity":"10",
"Route":"Intelligent",
"StopPrice":"0.00",
"Symbol":"GOOG",
"TradeAction":"Buy"
}
