TradeStation WebAPI

DataService : HotListDefinitions

Summary:¶

Search resource for available HotList requests. Can filter responses using OData protocol.

Details:¶

  • Method: GET
  • Path: /data/hotlistdefinitions
  • URI Parameters:
    • none

  • Authentication: Requires a valid access token.

  • Returns: Hot List Definition object

Errors:¶

  • 401 | Unauthorized
  • 5xx | Unknown internal service error. Contact TradeStation.

Examples:¶

Example Request: 
GET https://api.tradestation.com/v2/data/hotlistdefinitions?$filter=Activity%20eq%20'Rapidly%20Falling' HTTP/1.1
Authorization:  Bearer AccessTokenGoesHere
