TradeStation WebAPI

Users Service : Post Job Set

For Internal TradeStation Use Only¶

Summary:¶

Allows authenticated users to create a JobSet, a container for jobs(indicator/symbol pairs) that run on a remote server. If bar interval information is not explicitly set, the default bar interval will be 5 minutes.

Details:¶

  • Method: POST
  • Path: users/{userid}/Automation/jobsets
  • URI Parameters:
    • {userid}

  • Request Body:
    • Job Set

JSONP¶

  • URI Parameters
    • {userid}
  • Query String Parameters
    • data = URL Encoded JSON serialized Job Set object
    • callback = jQuery method to callback
    • oauth_token = AccessToken

  • URL: /jsonp/users/{userid}/automation/jobsets/create?data=EncodedSerializedJobSetObjectGoesHere&callback=jQueryCallback&oauth_token=AccessTokenGoesHere

  • Authentication: Currently only available to TradeStation. Requires a valid access token.

Returns:¶

JobSet Guid.

Errors:¶

  • 401 | Unauthorized
  • 5xx | Unknown internal service error. Contact TradeStation.
