TradeStation WebAPI

Users Service : Get User Job

Summary:¶

Returns a Job in a given JobSet belonging to a particular user. If not subscribed to, a JobSet's jobs will eventually be pruned from the repository.

Details:¶

  • Method: GET
  • Path: users/{userid}/Automation/JobSets/{JobSetGuid}/Jobs/{seqID}
  • URI Parameters:
    • {userid}
    • {JobSetGuid}
    • {seqID}

  • Authentication: Requires a valid access token.

Returns:¶

Job

Errors:¶

  • 401 | Unauthorized
  • 5xx | Unknown internal service error. Contact TradeStation.

Examples:¶

Request:¶

GET https://api.tradestation.com/v2/users/testuser/automation/JobSets/05432717-513a-4def-bdd8-31b431e40597/Jobs/2 HTTP/1.1
Accept: application/json
