TradeStation WebAPI

Users Service : Remove Job from Job Set

Summary:¶

Allows authenticated users to remove a Job(indicator/symbol pair) from a JobSet, a container for jobs.

Details:¶

  • Method: DELETE
  • Path: users/{userid}/Automation/jobsets/{jobSetGuid}/jobs/{jobSequenceIds}
  • URI Parameters:
    • {userid}
    • {jobSetGuid}
    • {jobSequenceIds} = job ids, comma separated.

  • Request Body:
    • None

JSONP¶

  • URI Parameters
    • {userid}
    • {jobSetGuid}
  • Query String Parameters
    • data = job sequence id's
    • callback = jQuery method to callback
    • oauth_token = AccessToken

  • URL: /jsonp/users/{userid}/automation/jobsets/{jobSetGuid}/Jobs/delete?data=EncodedCommaSeparatedJobSequenceIDsGoHere&callback=jQueryCallback&oauth_token=AccessTokenGoesHere

  • Authentication: Requires a valid access token.

Returns:¶

Boolean

Errors:¶

  • 401 | Unauthorized
  • 5xx | Unknown internal service error. Contact TradeStation.

Examples:¶

Example Request¶

DELETE https://api.qa.tradestation.com/v2/users/testUser/automation/jobsets/b9171d61-7c54-4875-98a7-655b4d0caf59/jobs/2
