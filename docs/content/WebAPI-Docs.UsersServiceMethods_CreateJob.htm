TradeStation WebAPI

Users Service : Add Job to Job Set

For Internal TradeStation Use Only¶

Summary:¶

Allows authenticated users to add a Job(indicator/symbol pair) to a JobSet, a container for jobs. If bar interval information is not explicitly set, the default bar interval will be 5 minutes.

Details:¶

  • Method: POST
  • Path: users/{userid}/Automation/jobsets/{jobSetGuid}/jobs
  • URI Parameters:
    • {userid}
    • {jobSet guid}

  • Request Body:
    • Collection of Job

JSONP¶

  • URI Parameters
    • {userid}
    • {jobSetGuid}
  • Query String Parameters
    • data = URL Encoded JSON serialized Job object
    • callback = jQuery method to callback
    • oauth_token = AccessToken

  • URL: /jsonp/users/{userid}/automation/jobsets/{jobSetGuid}/Jobs/create?data=EncodedSerializedJobObjectGoesHere&callback=jQueryCallback&oauth_token=AccessTokenGoesHere

  • Authentication: Requires a valid access token.

Returns:¶

Job Sequence ID.

Errors:¶

  • 401 | Unauthorized
  • 5xx | Unknown internal service error. Contact TradeStation.
