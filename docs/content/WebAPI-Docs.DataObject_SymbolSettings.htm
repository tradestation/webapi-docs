TradeStation WebAPI

DataObject : Symbol Settings
Field Type Description
Symbol string
BarSize int Set for indicators that are chartable
BarUnit Minute, Daily, Weekly, Monthly Set for indicators that are chartable
StartDate DateTime Set for indicators that are chartable
EndDate DateTime Set for indicators that are chartable
BarsBack int Set for indicators that are chartable

  1. A job with an EndDate of the current day before market close, will not stream to the end of market. It will return plots up until the current time.
  2. A job with an EndDate that is out of session, will not send any plots, Bug:33335
  3. A job with a StartDate that is out of session, will work if the indicator needs to pull additional bars back.
