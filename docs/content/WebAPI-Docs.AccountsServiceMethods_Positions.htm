TradeStation WebAPI

Accounts Service : Positions

Summary:¶

Requesting position information for a particular account

Details:¶

  • Method: GET
  • Path /accounts/{accountkey}/positions
  • URI Parameters:
    • {accountkey}

  • Authentication: Requires a valid access token.

Returns:¶

Position object

Results Filter:¶

This endpoint accomodates the Open Data Protocol which can be used to filter responses.

Example positions request that filters results for symbol = AAPL 
/v2/accounts/123456/positions?oauth_token=accessTokenGoesHere&$filter=symbol%20eq%20'AAPL'

Errors:¶

  • 401 | Unauthorized
  • 5xx | Unknown internal service error. Contact TradeStation.

Authorization: Bearer b0R4MHZ5WjhVUVBzQW5w
